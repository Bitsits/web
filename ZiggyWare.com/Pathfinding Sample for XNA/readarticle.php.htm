Pathfinding Sample for XNA

Pathfinding Sample for XNA

by Semei


Russian Translation

This sample covers pathfinding and movable path object class with smooth curve movement, using XNA Curve class.


Path Finding in 2D

Controls:
  • F - Delete Tiles (followed by left click on the map)
  • W - Draw Tiles (followed by left click on the map)
  • Left Mouse Button - Move character
  • +/- Change Heuristics value


The patfinding,is meant to be optimized. I used a nice priority queue made by Jim Mischel. You can find implementation details in code coments.

Download the 2D Path Finding Sample



Path Finding in 3D

Controls:
  • W, A, S, D - move camera
  • Hold right mouse button and drag - change camera view
  • Press o (to enable), then right mouse button on playfied - move character
  • Press Left mouse button and drag - draw or remove walls
  • B - toggle drawing/removing mode
